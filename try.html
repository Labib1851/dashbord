Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Content with HTML tags.

innerText: Only visible text.

textContent: All text (visible or hidden, but without HTML tags).

Question-3: What is event delegation in the DOM?

Event delegation is a technique where an event listener is added to a parent element, and through that, events on its child elements are handled. It improves performance and helps handle events on dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is the process where an event triggered on a child element propagates upward to its parent elements. This allows parent elements to listen for events on their child elements without needing individual listeners on each child. You can stop the event from bubbling further by using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements: Use document.createElement() to create new HTML elements.

Adding Elements: Use appendChild() or insertBefore() to add newly created elements to the DOM.

Removing Elements: Use remove() or removeChild() to remove an element from the DOM.
